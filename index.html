¿Cómo se calcula?

Volumen diario para pacientes con peso menor o igual a 30 kg:

  • De 0kg a 10kg, se calcula 100cc por cada kilo.
  • Se suman 50cc por cada kilo de peso por arriba de 10kg, hasta 20kg.
  • De 20kg para arriba, se suman 20cc por cada kilo adicional

El resultado obtenido se divide por 24 horas para determinar el flujo horario.

Para pacientes con peso mayor a 30 kg se emplea la fórmula:

((peso * 4) + 7)/(peso + 90)

Luego se multiplica por 1500 o 2000 para determinar el volumen diario
